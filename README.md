# CyberShield Security Center

A modern, responsive cybersecurity dashboard built with **HTML, CSS, and JavaScript**. CyberShield provides an interactive interface for simulating security operations, including URL analysis, password strength assessment, vulnerability reporting, security metrics, and visual analytics.

## 🌐 Live Demo

**View the application here:**

**https://boisterous-llama-7a6b81.netlify.app/**

---

## Features

* Modern Security Operations Center (SOC) dashboard
* Responsive enterprise-inspired interface
* URL Risk Analysis
* Password Strength Analyzer
* Vulnerability Assessment
* Security Score Dashboard
* Threat Timeline Charts
* Threat Distribution Visualization
* Live Security Console
* Scan Statistics
* Interactive Cards
* Smooth Animations
* Glass-inspired Dark Theme
* Font Awesome Icons
* Chart.js Visualizations
